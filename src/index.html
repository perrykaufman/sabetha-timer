Sabetha Timer

By Perry Kaufman

How it works

  • Calls out each canon both 10 seconds in advance and as they spawn.
  • Begins with a 5 second countdown by default to allow for time to tab back into the game.
  • Timer starts with an additional 2 seconds to account for the time it takes to run onto the boss platform.
